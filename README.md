# Barbearia SaaS

Sistema completo de agendamento para barbearias.

## Funcionalidades
- Multi-tenancy (Múltiplas barbearias)
- Agendamento Premium
- Gestão Financeira
- Sistema de Fidelidade
- Branding Personalizado (Logo, Banner, Cores)
- Gestão de Equipe e Horários

## Deploy
Este projeto está configurado para deploy automático na Vercel.
Última atualização: Trigger Vercel Deploy.
